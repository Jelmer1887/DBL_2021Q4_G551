A quick guide on how to use our visualisation tool

1. Uploading a .csv file

Simply click on the upload button on the navbar or on the "Upload your own csv" button! This can be done from any page.

2. Understanding the visualisation

Now that the file is uploaded, a force directed graph will be generated. A node is an employee, the thickness of an edge represents the number of emails exchanged and red or green edges represent positive or negative interactions.

3. Interaction techniques

To better understand the graph, you can drag any the nodes around, hover over a node to get more information about it or click a node or an edge to print information the console.
